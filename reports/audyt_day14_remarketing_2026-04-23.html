📧 Naprawa #3 — Klaviyo UTM (deep dive)

Stan obecny kampanii Klaviyo

Analiza 49 wysłanych kampanii email z Klaviyo API:

UstawienieKampanie%Uwagi
add_tracking_params: true, puste custom params 23 47% Używają Klaviyo defaults (mogą być puste!)
add_tracking_params: false 25 51% Brak UTM na linkach
add_tracking_params: true + custom UTM 1 2% Jedyna z jawnym utm_source=Klaviyo

Problem: 51% kampanii nie ma UTM w ogóle

25 z 49 kampanii wysłanych ma add_tracking_params: false — linki idą bez żadnych parametrów UTM. Klaviyo defaults (gdy custom_tracking_params jest puste) mogą, ale nie muszą dodawać UTM — zależy od ustawień konta w Klaviyo UI (Settings → UTM Tracking).

Opcje naprawy UTM w Klaviyo

Opcja A: Ustawienia globalne w Klaviyo UI (REKOMENDOWANE)

W panelu Klaviyo: Settings → UTM Tracking → Enable UTM parameters.
Ustawić domyślne wartości:

  • utm_source = klaviyo
  • utm_medium = email
  • utm_campaign = {{ campaign.name }} (dynamiczny)

Dotyczy WSZYSTKICH nowych kampanii i flows automatycznie. Najszybsza opcja — 2 minuty w UI.

Opcja B: Programatycznie przez Klaviyo MCP API

Przy tworzeniu kampanii z create_campaign można ustawić tracking_options z custom_tracking_params:

"tracking_options": { "value": { "add_tracking_params": true, "custom_tracking_params": [ { "type": "static", "name": "utm_source", "value": "klaviyo" }, { "type": "static", "name": "utm_medium", "value": "email" }, { "type": "dynamic", "name": "utm_campaign", "value": "campaign_name" } ], "is_tracking_clicks": true, "is_tracking_opens": true } }

Dynamiczne wartości dostępne: campaign_id, campaign_name, campaign_name_id, campaign_name_send_day, email_subject, message_type, profile_id.

Opcja C: Ręczne UTM w linkach HTML szablonów

Dodawanie ?utm_source=klaviyo&utm_medium=email&utm_campaign=NAZWA do każdego linka w szablonie HTML. Najbardziej niezawodne, ale wymaga edycji każdego szablonu osobno. Można łączyć z Opcją A jako fallback.

REKOMENDACJA: Opcja A (globalne defaults) + Opcja B (API dla nowych kampanii tworzonych przez MCP). Dzięki temu WSZYSTKIE przyszłe emaile będą miały spójne UTM, a kampanie tworzone przez nas automatycznie dostaną prawidłowe tracking params.
