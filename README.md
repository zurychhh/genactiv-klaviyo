## Raporty i deliverables (HTML/MD na top-levelu)

Pliki `.html` to **samodzielne raporty** — otwierasz je dwuklikiem w dowolnej przeglądarce, bez logowania i bez żadnego serwera.

|------|------------|
| **[`benchmark-email-marketing-5-marek.html`](benchmark-email-marketing-5-marek.html)** | Benchmark e-mail marketingu: 5 marek DTC (Seed, ARMRA, IM8, Spacegoods, AG1) zestawionych z naszymi flow Klaviyo. Zakładki: **Rekomendacje** (8 bloków z gotowymi promptami do Claude Code / Claude design) + osobna zakładka na każdą markę z pełną listą maili. Próba: ~94 maile, maj–czerwiec 2026. |
| [`audyt-remarketing-google-ads.html`](audyt-remarketing-google-ads.html) | Audyt remarketingu Google Ads. |
| [`BACKLOG_KLIENT_Q2_2026.html`](BACKLOG_KLIENT_Q2_2026.html), [`BACKLOG_PRIORYTETYZACJA_Q2_2026.html`](BACKLOG_PRIORYTETYZACJA_Q2_2026.html) | Backlog i priorytetyzacja Q2 2026. |
| [`black-friday-2024-2025-report.md`](black-friday-2024-2025-report.md) | Raport Black Friday 2024/2025. |
